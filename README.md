# Sales Forecasting

Predicting daily retail sales using time series techniques.

## Challenge
- 1,115 stores × 2.5 years of daily data
- Seasonality (weekends, holidays, months)
- Temporal dependencies (yesterday affects today)
- Can't shuffle - time order matters
